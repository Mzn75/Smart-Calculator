# Arabic Smart Calculator by Mz
- An intelligent calculator designed to help students with **numbers, geometric shapes, and mathematical theorems** — all in Arabic for easier learning.
- You can run the program using the `.exe` file

## Features
- Perform basic and advanced calculations in Arabic  
- Work with **numbers** 
- Handle **shapes** (geometry formulas, area, perimeter)  
- Helps in applying **theorems** step by step  
- User‑friendly interface designed for students
